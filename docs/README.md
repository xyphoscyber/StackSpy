# docs/

This folder contains documentation assets for StackSpy.

## screenshot.png

Referenced in the README. To generate:
1. Load the extension in Chrome developer mode
2. Visit a site with a known database stack (e.g. a Firebase app, a Django site)
3. Open the StackSpy popup
4. Take a screenshot and save it here as `screenshot.png` (recommended: 800×500px)
